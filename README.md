# Codding Puzzles

A personal monorepo for practicing coding puzzles while learning to code.

## Structure

- `advent-of-code/` - Advent of Code solutions by year and day.
- `leetcode/` - LeetCode solutions grouped by topic or difficulty.
- `notes/` - Learning notes, patterns, and reminders.

## Suggested Workflow

1. Create a folder for each problem.
2. Add the solution file and a short `README.md` explaining the approach.
3. Commit after each solved problem.

Example:

```text
leetcode/
  arrays/
    two-sum/
      README.md
      solution.php
```

## Languages

This repository uses PHP for coding-puzzle solutions.
